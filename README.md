### Run Locally (OS X)
1. Make sure Brew is isntalled and the latest version of Ruby (2.4.2)
2. cd into/the/location/of/the/project
3. Install rails  
  `$ gem install rails; hash -r; rails -v;`  
4. Install Postgres  
  `$ brew update; brew install postgres`  
5. Install gems  
  `$ bundle install`
6. Launch the server  
  `$ rails server`
7. Visit the site  
  `http://localhost:3000/`
